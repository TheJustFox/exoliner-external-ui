# Exoliner External UI
An TOS-allowed executor made for exoliner.wtf

# Features
- Tab system
- Colored editor (using FastColoredTextbox)
- File save/load
- Script hub
- Auto execute
- Shortcuts

# How to use
Step 1: build or download release
Step 2: open main exe
Step 3: log in using JWT cookie
